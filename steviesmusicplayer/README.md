# Stevie's One-Button Music Player

A simple web application designed for easy music playback.

## Features
- **One-Button Control**: Press the **Spacebar** or click the big button to play a random song from the playlist.
- **Shuffle**: Automatically picks a random song each time.
- **Edit Playlist**: Parents can easily add or remove YouTube links via the "Edit Playlist" button.
- **Auto-Save**: Your playlist is saved automatically in your browser, so it remembers your songs next time you open it.

## How to Use
1. Open `index.html` in your web browser (Chrome, Edge, Firefox, etc.).
2. Press the **Spacebar** to start playing music.
3. Press **Spacebar** again at any time to skip to a random next song.
4. To change the songs:
   - Click the **Edit Playlist** button at the bottom.
   - Paste YouTube links (one per line) into the box.
   - Click **Save Playlist**.

## Setup
No installation required. Just keep `index.html`, `style.css`, and `script.js` in the same folder.
